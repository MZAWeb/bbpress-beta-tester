== Description ==

Lets you easily upgrade to the latest trunk version of bbPress from your WordPress admin.

== Installation ==

1. Place the 'bbpress-beta-tester' folder in your '/wp-content/plugins/' directory.
2. Activate it.
3. Go to Dashboard->Upgrade or to the plugins screen
